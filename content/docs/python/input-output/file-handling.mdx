## Aprire un file: `open()`

Per lavorare con i file, si usa la funzione `open()`. Restituisce un **oggetto file** con cui puoi leggere o scrivere.

```python
file = open("nome_file.txt", "modalità")
```

Le modalità principali:

| Modalità | Descrizione |
|----------|-------------|
| `"r"` | Lettura (default). Errore se il file non esiste. |
| `"w"` | Scrittura. Crea il file se non esiste, lo svuota se esiste. |
| `"a"` | Aggiunta. Aggiunge alla fine senza cancellare il contenuto. |
| `"x"` | Creazione esclusiva. Errore se il file esiste già. |
| `"r+"` | Lettura e scrittura. |
| `"b"` | Modalità binaria (es. `"rb"`, `"wb"`). |

## Leggere un file

### Leggere tutto il contenuto

```python
file = open("esempio.txt", "r")
contenuto = file.read()
print(contenuto)
file.close()
```

### Leggere riga per riga

```python
file = open("esempio.txt", "r")
for riga in file:
    print(riga, end="")  # riga include già \n
file.close()
```

### Leggere come lista di righe

```python
file = open("esempio.txt", "r")
righe = file.readlines()   # lista di stringhe, una per riga
print(righe)
file.close()
```

## Il modo corretto: `with`

È **fortemente consigliato** usare il costrutto `with`, che chiude automaticamente il file anche se si verifica un errore:

```python
with open("esempio.txt", "r") as file:
    contenuto = file.read()
    print(contenuto)
# Il file viene chiuso automaticamente qui
```

```python
with open("esempio.txt", "r") as file:
    for riga in file:
        print(riga.strip())  # strip() rimuove \n e spazi
```

## Scrivere su un file

### Creare/sovrascrivere un file

```python
with open("output.txt", "w") as file:
    file.write("Prima riga\n")
    file.write("Seconda riga\n")
    file.write("Terza riga\n")
```

### Scrivere più righe con `writelines()`

```python
righe = ["Prima riga\n", "Seconda riga\n", "Terza riga\n"]

with open("output.txt", "w") as file:
    file.writelines(righe)
```

### Aggiungere a un file esistente

```python
with open("output.txt", "a") as file:
    file.write("Riga aggiunta\n")
```

## Verificare se un file esiste

```python
import os

if os.path.exists("esempio.txt"):
    print("Il file esiste")
else:
    print("Il file non esiste")
```

## Gestire gli errori con i file

```python
try:
    with open("file_inesistente.txt", "r") as file:
        contenuto = file.read()
except FileNotFoundError:
    print("Errore: il file non esiste.")
except PermissionError:
    print("Errore: non hai i permessi per aprire il file.")
```

## Esempio pratico: registro studenti

```python
def salva_studenti(studenti, filename):
    with open(filename, "w") as file:
        for nome, voto in studenti:
            file.write(f"{nome},{voto}\n")

def carica_studenti(filename):
    studenti = []
    try:
        with open(filename, "r") as file:
            for riga in file:
                parti = riga.strip().split(",")
                if len(parti) == 2:
                    nome = parti[0]
                    voto = float(parti[1])
                    studenti.append((nome, voto))
    except FileNotFoundError:
        print("File non trovato, parto con lista vuota.")
    return studenti

# Salva i dati
studenti = [("Alice", 8.5), ("Bob", 7.0), ("Carlo", 9.2)]
salva_studenti(studenti, "studenti.csv")

# Ricarica i dati
caricati = carica_studenti("studenti.csv")
for nome, voto in caricati:
    print(f"{nome}: {voto}")
```

## Cos'è un ciclo?

Un **ciclo** permette di ripetere un blocco di codice più volte. Senza i cicli, dovresti scrivere la stessa istruzione decine di volte.

## Il ciclo `while`

Il ciclo `while` ripete il blocco di codice **finché la condizione rimane vera**:

```python
while condizione:
    # codice da ripetere
```

```python
contatore = 1

while contatore <= 5:
    print(contatore)
    contatore += 1
```

Output:
```
1
2
3
4
5
```

Il ciclo continua finché `contatore <= 5` è vero. A ogni iterazione, `contatore` aumenta di 1. Quando arriva a 6, la condizione diventa falsa e il ciclo si ferma.

## Attenzione al ciclo infinito

Se la condizione non diventa mai falsa, il ciclo non finisce mai (ciclo infinito). Questo blocca il programma!

```python
# ATTENZIONE: questo è un ciclo infinito, non eseguirlo!
x = 0
while x < 10:
    print(x)
    # dimenticato: x += 1
```

Per interrompere un ciclo infinito, usa `Ctrl + C` nel terminale.

## L'istruzione `break`

`break` interrompe il ciclo immediatamente, anche se la condizione è ancora vera:

```python
contatore = 0

while True:  # ciclo potenzialmente infinito
    print(contatore)
    contatore += 1
    if contatore == 5:
        break  # esci quando contatore raggiunge 5
```

Output:
```
0
1
2
3
4
```

## L'istruzione `continue`

`continue` salta il resto del codice nell'iterazione corrente e passa alla prossima:

```python
contatore = 0

while contatore < 10:
    contatore += 1
    if contatore % 2 == 0:  # se il numero è pari
        continue             # salta la stampa
    print(contatore)         # stampa solo i dispari
```

Output:
```
1
3
5
7
9
```

## Il blocco `else`

Un ciclo `while` può avere un blocco `else`, che viene eseguito quando il ciclo termina normalmente (senza `break`):

```python
contatore = 0

while contatore < 3:
    print(contatore)
    contatore += 1
else:
    print("Ciclo terminato normalmente!")
```

Output:
```
0
1
2
Ciclo terminato normalmente!
```

Se il ciclo termina con `break`, il blocco `else` non viene eseguito:

```python
contatore = 0

while contatore < 10:
    if contatore == 3:
        break
    print(contatore)
    contatore += 1
else:
    print("Questo non viene stampato")
```

## Esempi pratici

### Chiedere input finché non è valido

```python
while True:
    risposta = input("Inserisci 'sì' o 'no': ")
    if risposta == "sì" or risposta == "no":
        break
    print("Input non valido, riprova.")

print(f"Hai scelto: {risposta}")
```

### Conto alla rovescia

```python
n = 5

while n > 0:
    print(n)
    n -= 1

print("Via!")
```

## Cosa sono le tuple?

Una **tupla** (`tuple`) è simile a una lista: è una raccolta ordinata di elementi. La differenza principale è che le tuple sono **immutabili**: una volta create, non si possono modificare.

## Creare una tupla

Le tuple si scrivono con le **parentesi tonde** `()`:

```python
coordinate = (10, 20)
colori = ("rosso", "verde", "blu")
mista = (1, "ciao", True, 3.14)
vuota = ()
```

Puoi anche creare una tupla senza parentesi (ma è meno leggibile):

```python
giorni = "lun", "mar", "mer"
print(type(giorni))  # <class 'tuple'>
```

**Tupla con un solo elemento:** devi aggiungere una virgola finale, altrimenti Python lo interpreta come una semplice espressione tra parentesi:

```python
non_e_tupla = (42)
print(type(non_e_tupla))   # <class 'int'>

e_una_tupla = (42,)
print(type(e_una_tupla))   # <class 'tuple'>
```

## Accedere agli elementi

L'accesso funziona esattamente come per le liste, con gli indici:

```python
colori = ("rosso", "verde", "blu")
print(colori[0])    # rosso
print(colori[-1])   # blu
print(colori[1:3])  # ('verde', 'blu')
```

## Immutabilità

Tentare di modificare una tupla causa un errore:

```python
colori = ("rosso", "verde", "blu")
colori[0] = "giallo"  # TypeError: 'tuple' object does not support item assignment
```

## Metodi delle tuple

Le tuple hanno meno metodi delle liste, proprio perché non sono modificabili:

```python
numeri = (1, 2, 3, 2, 4, 2)
print(numeri.count(2))   # 3
print(numeri.index(3))   # 2
print(len(numeri))       # 6
```

## Operazioni sulle tuple

```python
a = (1, 2, 3)
b = (4, 5, 6)

print(a + b)    # (1, 2, 3, 4, 5, 6)  (concatenazione)
print(a * 2)    # (1, 2, 3, 1, 2, 3)  (ripetizione)
print(3 in a)   # True
print(7 in a)   # False
```

## Conversione tra tuple e liste

```python
lista = [1, 2, 3]
tupla = tuple(lista)
print(tupla)          # (1, 2, 3)

di_nuovo = list(tupla)
print(di_nuovo)       # [1, 2, 3]
```

## Tuple unpacking

Puoi assegnare gli elementi di una tupla a più variabili in una sola riga:

```python
coordinate = (10, 20)
x, y = coordinate
print(x)  # 10
print(y)  # 20

nome, eta, citta = ("Alice", 16, "Roma")
print(nome)   # Alice
print(eta)    # 16
print(citta)  # Roma
```

Questo funziona anche con le liste.

## Quando usare una tupla invece di una lista?

Usa una **tupla** quando:
- I dati non devono cambiare (es. coordinate, giorni della settimana, colori fissi)
- Vuoi usarla come chiave di un dizionario (le liste non si possono usare come chiavi)
- Vuoi comunicare che quei dati sono "sola lettura"

Usa una **lista** quando hai bisogno di aggiungere, rimuovere o modificare elementi.

Le tuple sono anche leggermente più veloci delle liste.

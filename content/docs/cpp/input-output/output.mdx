## Cos'è `cout`?

`cout` (pronuncia "si-aut", da "character output") è lo strumento principale del C++ per visualizzare output sullo schermo. Fa parte della libreria `<iostream>` e del namespace `std`.

Per usarlo, devi includere `<iostream>` all'inizio del file:

```cpp
#include <iostream>
using namespace std;
```

## L'operatore `<<`

L'operatore `<<` si chiama **operatore di inserimento**. Serve a "inviare" dati verso `cout`:

```cpp
cout << "Ciao, Mondo!";
```

Puoi immaginarlo come una freccia che punta verso destra: stai inserendo il testo nel flusso di output.

## Stampa di testo

Per stampare del testo (una stringa letterale), mettilo tra virgolette doppie:

```cpp
cout << "Ciao!";
cout << "Questa è una frase.";
```

## Andare a capo con `endl` e `\n`

Esistono due modi per andare a capo:

```cpp
cout << "Prima riga" << endl;    // usa endl
cout << "Seconda riga\n";        // usa \n all'interno della stringa
```

Entrambi vanno a capo. La differenza è che `endl` svuota anche il buffer di output (più lento ma più sicuro in certi contesti). Per la maggior parte dei programmi semplici, sono equivalenti.

## Concatenare con `<<`

Puoi usare più `<<` nella stessa istruzione per stampare più cose:

```cpp
cout << "Nome: " << "Alice" << endl;
cout << "Eta: " << 16 << endl;
cout << "Altezza: " << 1.65 << " m" << endl;
```

Output:

```
Nome: Alice
Eta: 16
Altezza: 1.65 m
```

## Stampare variabili

`cout` funziona con qualsiasi tipo di dato:

```cpp
int x = 42;
double pi = 3.14;
char lettera = 'A';
bool vero = true;

cout << x << endl;        // 42
cout << pi << endl;       // 3.14
cout << lettera << endl;  // A
cout << vero << endl;     // 1 (true viene stampato come 1)
```

## Caratteri speciali (sequenze di escape)

All'interno delle stringhe puoi usare caratteri speciali preceduti da `\`:

| Sequenza | Significato |
|---|---|
| `\n` | Nuova riga |
| `\t` | Tabulazione (tab) |
| `\\` | Backslash (`\`) |
| `\"` | Virgolette doppie (`"`) |
| `\'` | Virgoletta singola (`'`) |

```cpp
cout << "Riga 1\nRiga 2\nRiga 3" << endl;
cout << "Nome:\tAlice" << endl;
cout << "Dice: \"Ciao!\"" << endl;
```

Output:

```
Riga 1
Riga 2
Riga 3
Nome:   Alice
Dice: "Ciao!"
```

## Calcoli nell'output

Puoi eseguire calcoli direttamente nell'output:

```cpp
int a = 5, b = 3;
cout << "Somma: " << a + b << endl;
cout << "Prodotto: " << a * b << endl;
```

Output:

```
Somma: 8
Prodotto: 15
```

## Stampare su più righe

```cpp
cout << "Prima riga" << endl;
cout << "Seconda riga" << endl;
cout << "Terza riga" << endl;
```

Oppure con `\n`:

```cpp
cout << "Prima riga\nSeconda riga\nTerza riga\n";
```

Entrambi producono lo stesso output:

```
Prima riga
Seconda riga
Terza riga
```

## Esempio pratico

```cpp
#include <iostream>
using namespace std;

int main() {
    string nome = "Alice";
    int eta = 16;
    double altezza = 1.65;

    cout << "=== Scheda Studente ===" << endl;
    cout << "Nome:    " << nome << endl;
    cout << "Eta:     " << eta << " anni" << endl;
    cout << "Altezza: " << altezza << " m" << endl;

    return 0;
}
```

Output:

```
=== Scheda Studente ===
Nome:    Alice
Eta:     16 anni
Altezza: 1.65 m
```

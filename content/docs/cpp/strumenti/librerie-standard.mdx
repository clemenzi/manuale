## Cos'è la libreria standard?

La **libreria standard del C++** (Standard Library) è un insieme di librerie già pronte che vengono fornite con ogni compilatore C++. Contengono funzioni, classi e costanti per le operazioni più comuni: input/output, matematica, stringhe, contenitori, algoritmi, e molto altro.

Per usare una libreria, la includi con `#include`.

## `<iostream>` - Input e Output

La libreria fondamentale per input e output:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Output su schermo" << endl;
    cerr << "Messaggio di errore" << endl;
    
    int x;
    cin >> x;
    
    return 0;
}
```

Principali oggetti:
- `cout`: output su schermo
- `cin`: input da tastiera
- `cerr`: output degli errori
- `endl`: vai a capo e svuota il buffer

## `<string>` - Stringhe

Per usare il tipo `string`:

```cpp
#include <string>
using namespace std;

string nome = "Alice";
int lunghezza = nome.length();
string maiuscola = "";
for (char c : nome) {
    maiuscola += toupper(c);
}
cout << maiuscola << endl;   // ALICE
```

## `<cmath>` - Funzioni Matematiche

Contiene funzioni matematiche avanzate:

```cpp
#include <cmath>
using namespace std;

// Potenza
cout << pow(2, 10) << endl;     // 1024 (2^10)
cout << pow(3, 3) << endl;      // 27

// Radice quadrata
cout << sqrt(25) << endl;       // 5
cout << sqrt(2) << endl;        // 1.41421

// Valore assoluto
cout << abs(-5) << endl;        // 5
cout << fabs(-3.14) << endl;    // 3.14 (per double)

// Arrotondamento
cout << floor(3.7) << endl;     // 3 (arrotonda in giù)
cout << ceil(3.2) << endl;      // 4 (arrotonda in su)
cout << round(3.5) << endl;     // 4 (arrotonda al più vicino)

// Funzioni trigonometriche (in radianti)
cout << sin(0) << endl;         // 0
cout << cos(0) << endl;         // 1

// Logaritmo
cout << log(M_E) << endl;       // 1 (logaritmo naturale)
cout << log10(1000) << endl;    // 3

// Costante PI
cout << M_PI << endl;           // 3.14159...
```

## `<cstdlib>` - Funzioni di utilità generali

```cpp
#include <cstdlib>
#include <ctime>
using namespace std;

// Numeri casuali
srand(time(0));   // inizializza il generatore
int casuale = rand() % 100;   // numero tra 0 e 99
int dado = rand() % 6 + 1;   // numero tra 1 e 6

// Conversione stringa -> numero (alternativa a stoi/stod)
int n = atoi("42");
double d = atof("3.14");

// Uscita dal programma
// exit(0);    // esce con successo
// exit(1);    // esce con errore
```

## `<cctype>` - Funzioni sui caratteri

```cpp
#include <cctype>
using namespace std;

char c = 'A';
cout << isupper(c) << endl;    // 1 (true): è maiuscolo?
cout << islower(c) << endl;    // 0 (false): è minuscolo?
cout << isdigit('5') << endl;  // 1: è una cifra?
cout << isalpha('x') << endl;  // 1: è una lettera?
cout << isspace(' ') << endl;  // 1: è uno spazio?

cout << tolower('A') << endl;  // 'a': converti in minuscolo
cout << toupper('a') << endl;  // 'A': converti in maiuscolo
```

## `<algorithm>` - Algoritmi

Contiene algoritmi già implementati per lavorare su sequenze:

```cpp
#include <algorithm>
#include <vector>
using namespace std;

vector<int> v = {5, 2, 8, 1, 9, 3};

// Ordinamento
sort(v.begin(), v.end());
// v è ora: {1, 2, 3, 5, 8, 9}

// Ordinamento decrescente
sort(v.begin(), v.end(), greater<int>());
// v è ora: {9, 8, 5, 3, 2, 1}

// Valore minimo e massimo
cout << *min_element(v.begin(), v.end()) << endl;  // 1
cout << *max_element(v.begin(), v.end()) << endl;  // 9

// Ricerca
auto it = find(v.begin(), v.end(), 5);
if (it != v.end()) {
    cout << "Trovato!" << endl;
}

// Inversione
reverse(v.begin(), v.end());
```

## `<vector>` - Vettori

Array dinamici (vedi sezione dedicata):

```cpp
#include <vector>
using namespace std;

vector<int> v = {1, 2, 3};
v.push_back(4);
v.push_back(5);
cout << v.size() << endl;   // 5
```

## `<limits>` - Limiti dei tipi

```cpp
#include <limits>
using namespace std;

cout << numeric_limits<int>::max() << endl;    // 2147483647
cout << numeric_limits<int>::min() << endl;    // -2147483648
cout << numeric_limits<double>::max() << endl; // 1.79769e+308
```

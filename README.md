# Manuale

Manuale è un sito di documentazione didattica costruito con Astro e Starlight. Raccoglie percorsi progressivi su **Python**, **SQL** e **C++**, con spiegazioni brevi, esempi leggibili e una struttura pensata per studenti e principianti assoluti.

Il progetto contiene oltre 160 pagine MDX organizzate per argomento, dalla sintassi di base fino a temi più avanzati come strutture dati, query SQL, transazioni, puntatori, programmazione a oggetti e strumenti di debugging.

> [!NOTE]
> I contenuti sono scritti in italiano e privilegiano chiarezza, gradualità e pratica rispetto alla completezza enciclopedica.
